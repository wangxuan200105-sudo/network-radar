# Privacy

Network Radar is an instruction-and-script package. The repository does not include a hosted service, telemetry, analytics, contact database, or credential collection.

When the Skill runs, it may process profile pages, search results, contact spreadsheets, and a user-selected Feishu Bitable that the user has authorized Codex to access. Those materials can contain personal information. Users should:

- collect only data needed for a legitimate networking purpose;
- follow applicable platform rules, laws, and organizational policies;
- avoid committing contact exports, browser sessions, tokens, or private notes to Git;
- review generated outreach before sending it;
- delete local outputs when they are no longer needed.

Network Radar does not instruct Codex to bypass login, CAPTCHA, access controls, or rate limits, and it never authorizes automatic outreach.
