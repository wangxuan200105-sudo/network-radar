# Contributing

Issues and pull requests are welcome.

Before submitting a change:

1. Keep `SKILL.md` concise and move conditional detail into `references/`.
2. Do not add real contact data, profile exports, tokens, cookies, screenshots containing personal information, or generated workbooks.
3. Preserve the strict profile-section evidence rule.
4. Run `python3 scripts/validate_repository.py --release`.
5. Explain behavior changes and include a small synthetic example when relevant.
